1
물류데이터 기반
PortWatch, MarineTraffic 등
→ 선박 AIS 이동 데이터 기반 물동량 추적
✕ 산업별 파급효과 미추적
✕ 국내 전파 경로 식별 불가
2
텍스트 기반
감성분석, 토픽모델링
→ 뉴스 텍스트에서 감성지수 도출
✕ 구체적 전파 경로 미식별
✕ 정량적 영향도 산출 한계
3
LLM 기반
GPT 에이전트, RAG
→ 범용 LLM으로 위기 분류·요약
✕ 유사 위기 간 혼동
